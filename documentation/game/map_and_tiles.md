# Map and Tiles

The solo map is a radius-based hex grid generated by the backend game engine.

Each tile stores:

- axial coordinates `q`, `r`
- `terrain`: `neutral`, `forest`, or `rock`
- `hydration`: integer clamped by terrain rules
- `building` and `building_upgrade`
- `stress` and `terrain_stress`
- optional `nutrient_type`: `green`, `blue`, `purple`, or `null`
- `extraction_progress`, used by Assimilators

Initial generation places the Core at the configured start tile, then places rocks, forests, moist tiles, dry tiles, and nutrient deposits. Nutrients are static map features and are not consumed.

Current nutrient tuning:

- 3 Green nutrient tiles
- 3 Blue nutrient tiles
- 3 Purple nutrient tiles
- placed away from the Core
- not placed on the Core, rocks, or existing buildings
